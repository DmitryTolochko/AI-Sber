# AI-Sber
«Цифровой хранитель: AI для сохранения и изучения малых языков России»

## Описание
Десятки языков коренных народов России находятся на грани исчезновения. Уходят носители, а вместе с ними - уникальные культурные коды, фольклор, знания. Существующие цифровые инструменты (синтез речи, перевод) для таких языков отсутствуют, что ускоряет их вытеснение и затрудняет изучение новым поколением.

## Цель
Разработать AI-платформу для поддержки и "оживления" одного из малых языков России (например, горномарийского, эвенкийского, ительменского), создав базовые инструменты для его использования в цифровой среде.

## Требуемый результат
Веб-сайт, посвященный выбранному языку.

На сайте представлены интерактивные инструменты:
- "Говорящий словарь": пользователь вводит слово на русском или на малом языке, система показывает перевод и озвучивает его на малом языке (Text-to-Speech).
- "Переводчик": прототип нейросетевого переводчика для простых фраз с русского на малый язык и обратно.
- "Чтец сказок": нейросеть зачитывает вслух небольшие тексты (сказки, пословицы), помогая в изучении произношения.
